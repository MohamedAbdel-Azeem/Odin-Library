# Odin-Library# Odin-Library

Odin-Library is a simple web application for managing a collection of books.
It is part of The Odin Project's curriculum and is the first project in the JavaScript section.

## Access the Website
You Can Access The Website via (https://mohamedabdel-azeem.github.io/Odin-Library/)

## Usage

- Click the "Add Book" button to add a new book to the library.
- Fill in the book details in the form and click "Submit".
- The new book will appear in the library.
- Click the "Remove" button on a book to remove it from the library.


## Note
There is Currently no Storage in the Web App.
